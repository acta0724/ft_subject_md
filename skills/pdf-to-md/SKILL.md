引数で指定されたPDFファイル（42の課題）を読み込み、内容を整形してMarkdownファイルとして保存してください。

## 手順

1. `$ARGUMENTS` で受け取ったパスのPDFファイルをReadツールで読み込む
2. 以下の方針でMarkdownに変換する：
   - タイトル・章立てを `#` `##` `###` で表現
   - コードブロックは ` ``` ` で囲む
   - 表は Markdown テーブル形式で書く
   - 42特有のセクション（Introduction, Goals, General Instructions, Mandatory Part, Bonus Part, Submission）を維持する
   - 数式はそのまま記述する
3. 出力ファイルは元PDFと同じディレクトリに `<元ファイル名>.md` として保存する
4. 保存後、変換結果のサマリを表示する

## 注意事項

- PDFの視覚的なレイアウト（ページ番号、ヘッダー/フッターなど）は除外する
- コードサンプルは正確に保持する
- リンクやメールアドレスは Markdown のリンク形式に変換する
