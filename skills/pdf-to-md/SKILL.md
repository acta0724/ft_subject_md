引数で指定されたパス（PDFファイルまたはディレクトリ）を受け取り、42の課題PDFをMarkdownファイルとして保存してください。

## 手順

1. `$ARGUMENTS` で受け取ったパスを判定する
   - **ファイルの場合**: そのPDF1件を処理する
   - **ディレクトリの場合**: Globツールで `<ディレクトリ>/**/*.pdf` を検索し、見つかった全PDFを処理する

2. 各PDFについて以下を行う：
   - Readツールでファイルを読み込む
   - 以下の方針でMarkdownに変換する：
     - タイトル・章立てを `#` `##` `###` で表現
     - コードブロックは ` ``` ` で囲む
     - 表は Markdown テーブル形式で書く
     - 42特有のセクション（Introduction, Goals, General Instructions, Mandatory Part, Bonus Part, Submission）を維持する
     - 数式はそのまま記述する
   - 出力ファイルは元PDFと同じディレクトリに `<元ファイル名>.md` として保存する

3. 全件処理後、変換したファイルの一覧をサマリとして表示する

## 注意事項

- PDFの視覚的なレイアウト（ページ番号、ヘッダー/フッターなど）は除外する
- コードサンプルは正確に保持する
- リンクやメールアドレスは Markdown のリンク形式に変換する
- 既に同名の `.md` が存在する場合は上書きする
